# Fitbit Gallery Submission - 中文 (Mandarin Chinese)

## App Name / 应用名称

活力 (Huólì) - Vital

## Short Description / 简短描述

专为 Fitbit Versa 2 设计的极简数字表盘，以简洁现代的设计展示时间、日期和健康指标。

## Long Description / 详细描述 (复制此内容到描述文本框)

活力是一款专为 Fitbit Versa 2 设计的时尚简约表盘。它以清晰易读和简洁为设计重点，展示必要信息。表盘采用黑色背景，配以精心选择的字体和颜色强调，以实现最佳的可见性和电池效率。

主要功能：

- 大号粗体数字时间显示，易于阅读
- 当前日期缩写格式显示
- 实时心率监测
- 彩色编码电池指示器
- 简洁极简设计，高对比度
- 支持常亮显示

非常适合希望一目了然地获取必要信息而无需视觉干扰的用户。
